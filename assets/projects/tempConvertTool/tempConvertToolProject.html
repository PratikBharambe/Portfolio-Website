Temperture Converter

Desktop Application

Temperture Converter Tool is a desktop application created by me, that get the temperture from the user and convert it to the temperture chossen by the user.

Process :

Step 1 :Open the application.

Step 2 :Choose the convertion from the choice box which you want to perform.

Step 3 :Enter the value of temperture in the textfield provided just below the choice box.

Step 4 :Press the convert button below the textfield and the result box will appear with the converted value with it's unit.

If in case the user entered a invalid input then after hitting the convert button an error window comes up and tell the user to enter the valid input.
